basic text formating

use the example strong tag or example b tag to make something bold.

The difference between strong and b

The strong tag is an html semantic tag. there is a meaning behind using the strong tag what that meaning is, it tells the browser there is something important about the text inside that tag.

The b tag means nothing, its just bold text.

the example em tag and the example i tag

The em tag and the i tag, make the text inside of them italic. again, the difference ios that the em tag is a semantic tag, the i tag just turns text italic.

the u tag

the example u tag simpily underlines text.
